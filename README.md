# Burger Boss
Imaginary Burger site with a different look than your normal burger site.

**Link to project:** https://burgerboss.netlify.app/

## How It's Made:

**Tech used:** HTML, CSS

Static website for an imaginary burger shop. No frameworks used - all vanilla

## Optimizations
Here is a list of what should be optimized
+ Interactive hamburger menu on mobile
+ Create the other sites (links are dead at the moment)
+ Create a better system in CSS for organization
+ Make order section a caroussel
+ Redesign about us section. It is way too big and overwhelming

## Lessons Learned:

Worked a lot with background images for the front page. Took my own design from figma and made a 1:1 static replica. Tried to work with odd colors and a complementary colorscheme
